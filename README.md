# PAGINA DE CUPCAKES.
Bienvenidos al repositorio de mi pagina de Cupcakes! 
Este es un proyecto de página web estática diseñado para un negocio ficticio de cupcakes, centrado en la estética, la facilidad de uso y la capacidad de respuesta móvil.

# Características Principales.
- Diseño Moderno y Atractivo: Un diseño limpio y apetitoso, ideal para una tienda de repostería.
- Totalmente Responsivo: Implementado con Bootstrap 5 para asegurar que el sitio se vea perfecto en cualquier dispositivo, desde teléfonos móviles hasta computadoras de escritorio.
- Navegación Intuitiva: Barra de navegación fija con enlaces a secciones clave.
- Componentes Reutilizables: Uso de componentes estándar de Bootstrap como carruseles, tarjetas (cards) y modales para una experiencia de usuario rica.

# Tecnologías Utilizadas
- Este proyecto fue desarrollado utilizando las siguientes tecnologías:
- HTML5: Estructura base de la página.
- CSS3 (Estilos Personalizados): Pequeños ajustes de estilo para personalizar la apariencia.
- Bootstrap 5: Framework CSS y JavaScript para el diseño responsivo y los componentes interactivos.
